interface Neuron : Activity
Content copied to clipboard

The main element of the NeuralNetwork. Evolution is targeted to find the fittest Neuron where fit function is collaborative Feedback from other Neurons. Implementing that interface keep in mind that any implementation valid, but the ones targeted on increasing incoming Feedback (by "pleasing" neighbour neurons) have more chances to survive during Evolution. Before developing your Neuron implementation check NeuralNetwork pipeline documentation to understand order in which Neuron function called.
