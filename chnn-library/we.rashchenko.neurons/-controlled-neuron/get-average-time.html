fun getAverageTime(): Double
Content copied to clipboard

Get average runtime of that Neuron. That time includes all times of Neuron public functions. Note that time measured and averaged only for controlled time steps. So, rarely called Neuron functions such as forgetSource are even more rarely estimated leaving it with default value 0 as average time for that function. That may introduce some bias that neurons that have called rare functions will be considered as slower neurons that others. Also, that may introduce bias against older neurons compared with new one that never estimated rare functions time yet. To avoid the first bias try to make more steps controlled. For the second one you can use issue #13 to discuss how to solve that bias.
