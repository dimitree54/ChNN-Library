class ControlledNeuron(baseNeuron: Neuron) : Neuron

Wrapper for any other baseNeuron that reveals some of its statistics, for example averageActivity or averageTouchTime. That statistics can be used by NeuralNetworkController to calculate external Feedback for that Neuron
