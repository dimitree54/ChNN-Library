SupervisedNeuron

class SupervisedNeuron(externalActivity: HiddenActivity, baseNeuron: Neuron) : InputNeuron
Content copied to clipboard

Implementation of the InputNeuron that copies value of the internal baseNeuron ignoring the externalActivity. That externalActivity is used to calculate getInternalFeedback. Internal feedback is based on the mismatch of these internal and external activities. That InputNeuron is good for training, but does not allow external activity to initially appear at NeuralNetwork.
