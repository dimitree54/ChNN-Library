Calculates k nearest to the newPosition positions from allPositions

Return

Collection with k nearest neighbours of newPosition. For small allPositions with size < k+1 there maybe less than k neighbours in returned collection.

Parameters

newPosition

a new position, it should be in allPositions

allPositions

all the other positions, should contain newPosition
