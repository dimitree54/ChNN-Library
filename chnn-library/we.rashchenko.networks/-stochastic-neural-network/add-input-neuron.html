Add InputNeuron node to the graph.

Calling that function with some Neuron you notify the NeuralNetworkWithInput that the Neuron can change its state not only due to Neuron.touch or Neuron.update, but also spontaneously. So the NeuralNetworkWithInput on each tick should check that Neuron status.

Return

newly generated id for the added InputNeuron. Now that InputNeuron will be referred with that id in neuronIDs and connections
