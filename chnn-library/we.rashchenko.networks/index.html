Package we.rashchenko.networks

Classes for chaotic neural networks. Describes how neurons in the network communicate with each other, but not how new neurons created and wired (check networks builders for that). So far the main implementation is StochasticNeuralNetwork developed for efficient inference of the chaotic neural network with sparse activations.
