NeuralNetworkWithInput

interface NeuralNetworkWithInput : NeuralNetwork
Content copied to clipboard

NeuralNetwork with some Neuron-nodes specially marked as InputNeuron. NeuralNetworkWithInput should check InputNeurons state on each tick to not miss any InputNeuron state change initiated from outside the NeuralNetwork.

To understand what privileges InputNeuron has, check its documentation.
