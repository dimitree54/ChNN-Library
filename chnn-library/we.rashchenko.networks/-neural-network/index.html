interface NeuralNetwork : Ticking

NeuralNetwork is a directed graph with Neuron as nodes. However, for privacy reasons, all that neurons are hidden under their IDs. Based on these graph connections neurons "touch" each other (check Neuron documentation to understand what "touch" is) (source Neuron of the connection "touches" target Neuron of the connection). After the touch, there is a feedback collection stage where neurons estimate each other quality based on reversed connections (the target node of the connection sends feedback to the source node).
